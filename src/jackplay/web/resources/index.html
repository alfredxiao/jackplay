Jackplay - plays instrument!
"All work and no play makes Jack a dull boy." — So, let's have Jack play! 		What is Jackplay?
  • Jackplay is a tool for developers, testers to diagnose program execution at runtime.
  • It supports languages running on JVM.
  • You can trace method execution.
  • You can modify method body LIVE!
 How to use?
  • To trace a method, input a class name and method name and then click 'Play'!
 How it works!
  • Java Instrumentation
  • Live class redefinition
